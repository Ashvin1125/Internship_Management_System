# Internship Management System

A full-stack **Internship Management System** built using **ASP.NET Core (.NET 8 MVC)** and **Entity Framework Core** to manage internship records, students, and administrative workflows.

This system helps organizations and colleges manage internship details, track progress, and maintain structured internship data in a centralized platform. Internship management systems typically provide dashboards, CRUD operations, and tracking features to streamline intern administration. ([pwsc.in][1])

---

# Features

* Student Internship Management
* Create / Edit / Delete Internship Records
* Dashboard Overview
* MVC Architecture (.NET 8)
* Entity Framework Core (Code First)
* SQL Server Database
* Razor Views UI
* Bootstrap Responsive Layout
* Validation (Client + Server)
* CRUD Operations

# Installation & Setup

### 1. Clone Repository

```bash
git clone https://github.com/Ashvin1125/Internship_Management_System.git
```

### 2. Open Project

Open in:

* Visual Studio 2022
* Visual Studio Code

### 3. Update Database

Run:

```bash
dotnet ef database update
```

### 4. Run Project

```bash
dotnet run
```

OR press:

```
F5 in Visual Studio
```
